Welcome to Seed Framework

Seed is a portable game development framework, currently we support Windows, Linux, MacOSX and iOS, other platforms are being experimented to have full support, these includes Android and Web (using either WebGL, Flash or Google Native Client).

Some features include 2D and 3D with OpenGL/DirectX, optional threading support (you can disable any thread usage), asynchronous I/O, full listener based events, particle system, sprite system with atlas support and so on... Some planned and in development features include shaders, bone animation, render to texture, integrated scripting system and full game editor with live preview.

Currently 2.0 is under closed development on bitbucket until a stable build with all core features. It is a rewrite of the version 1.0 that was used in 15+ commercial casual games.
